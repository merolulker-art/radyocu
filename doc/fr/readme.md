# Radyocu

* Développeur: Erol Ulker
* Version: 1.4.0

Extension de lecteur radio légère et accessible pour NVDA.

## Fonctionnalités
* **Recherche par nom:** Accès à la base de données Radio-Browser.
* **Lecture par URL:** Collez des liens directs pour écouter.
* **Favoris:** Sauvegarde automatique des stations écoutées.
* **Support M3U:** Importez et exportez vos listes.

## Raccourcis
* **NVDA+Shift+Ctrl+Z:** Ouvrir l'interface.
* **F7:** Lecture.
* **F8:** Arrêt.
* **F5 / F6:** Volume.
* **F9 / F10:** Station Suivante / Précédente.
* **ESC:** Masquer (La radio continue en arrière-plan).
