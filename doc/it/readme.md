# Radyocu

* Sviluppatore: Erol Ulker
* Versione: 1.4.0

Componente aggiuntivo per radio leggero e accessibile per NVDA.

## Caratteristiche
* **Cerca per Nome:** Usa il database Radio-Browser.
* **Riproduci da URL:** Incolla link diretti.
* **Preferiti:** Salvataggio automatico.
* **Supporto M3U:** Importa ed esporta playlist.

## Scorciatoie
* **NVDA+Shift+Ctrl+Z:** Apri interfaccia.
* **F7:** Riproduci.
* **F8:** Stop.
* **F5 / F6:** Volume.
* **F9 / F10:** Successivo / Precedente.
* **ESC:** Nascondi (La radio continua in background).
