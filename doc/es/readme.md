# Radyocu

* Desarrollador: Erol Ulker
* Versión: 1.4.0

Complemento de radio ligero y accesible para NVDA.

## Características
* **Búsqueda por Nombre:** Base de datos global Radio-Browser.
* **Reproducir por URL:** Pegue enlaces directos.
* **Favoritos:** Guardado automático.
* **Soporte M3U:** Importar y exportar listas.

## Atajos
* **NVDA+Shift+Ctrl+Z:** Abrir interfaz.
* **F7:** Reproducir.
* **F8:** Detener.
* **F5 / F6:** Volumen.
* **F9 / F10:** Siguiente / Anterior.
* **ESC:** Ocultar (La radio sigue sonando).
