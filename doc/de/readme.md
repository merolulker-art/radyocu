# Radyocu

* Entwickler: Erol Ulker
* Version: 1.4.0

Leichtes und barrierefreies Radio-Add-on für NVDA.

## Funktionen
* **Namenssuche:** Zugriff auf tausende Sender über die Radio-Browser Datenbank.
* **URL-Wiedergabe:** Direkte Stream-Links einfügen und abspielen.
* **Favoriten:** Sender werden automatisch gespeichert.
* **M3U-Support:** Import und Export von Wiedergabelisten.

## Tastenkürzel
* **NVDA+Shift+Ctrl+Z:** Benutzeroberfläche öffnen.
* **F7:** Wiedergabe.
* **F8:** Stopp.
* **F5 / F6:** Lautstärke.
* **F9 / F10:** Nächster / Vorheriger Sender.
* **ESC:** Ausblenden (Radio spielt im Hintergrund weiter).
