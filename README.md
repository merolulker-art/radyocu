# Radyocu Eklentisi

Radyocu eklentisi, NVDA kullanıcıları için özel olarak tasarlanmış, erişilebilir bir internet radyosu oynatıcısıdır. En önemli özelliği, radyo sesinin NVDA sentezleyici sesinden bağımsız olmasıdır.

## Özellikler
* **Arama ve Dinleme:** "Radyo Ara" kutusuna radyo adını yazıp bekleyin, otomatik arama yapar. TAB ile listelere geçin, ENTER veya F7 ile dinleyin.
* **İçeri/Dışarı Aktarma:** Seçenekler menüsünden bilgisayarınızdaki .m3u dosyalarını yükleyebilir veya kendi listenizi .m3u formatında kaydedebilirsiniz.

## Kısayol Tuşları
Bu eklenti varsayılan olarak şu kısayolları kullanır:

* **NVDA + Shift + Ctrl + Z:** Eklentiyi Açar.
* **F5 / F6:** Radyo Sesi Arttır / Azalt (NVDA etkilenmez)
* **F7:** Oynat
* **F8:** Durdur
* **F9 / F10:** Sonraki / Önceki Radyo
* **ESC:** Gizle
